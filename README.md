
City Connect: Smart Reporting for Safer Communities

Status: Under development Event: Part of Hack O’Clock 2.0 (IILM University, Greater Noida)

Project Overview
City Connect is a real-time civic engagement platform designed to bridge the gap between citizens and local authorities. It empowers residents to report infrastructure issues like potholes and broken bridges while providing a high-priority SOS feature for life-threatening emergencies.

Key Features
Live Location Reporting: Interactive maps powered by Leaflet.js and OpenStreetMap.
Emergency SOS: Instant, real-time alerts for authorities (using simplified polling/meta-refresh).
Photo Evidence: Seamless image uploads for issue verification via Cloudinary.
Smart Dashboard: A centralized interface for authorities to manage and track civic reports.
Tech Stack
This project is built using a lightweight and fast-to-deploy stack:

Backend: [Flask] (Python)
Database: [SQLite]
Frontend: HTML5, CSS3, Vanilla JavaScript
Maps API: [Google Maps API] + [Leaflet.js] + OpenStreetMap
Cloud Storage: [Cloudinary] (for Image Management)
